## 프로젝트의 목표와 기능
1.1 프로젝트 목표
- 판매자와 구매자를 구별하여 판매자가 상품을 등록, 판매하며 구매자는 구매하는 서비스입니다.
- JavaScript를 사용하여 상품 등록, 결제, 상품에 대한 CRUD를 직접 구현해보는 프로젝트

1.2 구현하고자하는 기능
- 필수 구현 기능
  - 구매자 페이지 中
    - 로그인 페이지
    - 상품 목록 페이지
    - 장바구니 페이지
  
- 선택 구형 기능
  - 구매자 페이지 中
    - 회원 가입 페이지
    - 상품 상세 페이지
    - 주문/결제 페이지
  - 판매자 페이지
    - 로그인 페이지
    - 회원 가입 페이지
    - 상품 목록 페이지
    - 상품 상세 페이지
    - 판매자 센서 페이지
    - 상품 등록 페이지
